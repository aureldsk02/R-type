Introduction

Welcome to the documentation for R-TYPE. This document provides an overview of the project, its architecture, and how to use its various components.

What is our R-TYPE

R-TYPE is an arcade multiplayer game. The gameplay is a one of a space shooter, with at least two players going through waves of ennemies. The game runs on an ECS game engine, and the server-client communication is made possible by a binary UDP protocol.

Parts of the project

Key parts of R-TYPE include:

  • Part 1: The game engine.
  • Part 2: The server.
  • Part 3: The client. For more information about each part, see the dedicated page.
