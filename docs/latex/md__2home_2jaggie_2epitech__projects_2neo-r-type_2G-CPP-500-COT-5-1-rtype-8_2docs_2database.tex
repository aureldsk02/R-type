The R-\/\+Type project uses {\bfseries{SQLite3}} as an embedded database to manage player accounts, authentication, and gameplay statistics. ~\newline
 SQLite was chosen because it is\+:
\begin{DoxyItemize}
\item {\bfseries{Lightweight and self-\/contained}} (no external server)
\item {\bfseries{Cross-\/platform}} (works perfectly on Windows \& Linux)
\item {\bfseries{Fast}} for local read/write operations
\item {\bfseries{Easy to integrate}} via {\bfseries{vcpkg}}
\item {\bfseries{Reliable and persistent}} even in offline environments
\end{DoxyItemize}

This design ensures the game remains {\bfseries{fully deployable}}, even in {\bfseries{low-\/connectivity or resource-\/limited setups}}, which is especially important in many African development contexts.
