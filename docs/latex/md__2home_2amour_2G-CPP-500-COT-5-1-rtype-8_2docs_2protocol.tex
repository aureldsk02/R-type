Each client hold a {\bfseries{session}} identified by an UUID 128-\/bits. The possible state are \+:


\begin{DoxyItemize}
\item {\ttfamily DISCONNECTED}\+: Not connected
\item {\ttfamily CONNECTING}\+: TCP connection in progress
\item {\ttfamily AUTHENTICATING}\+: Authentication in progress
\item {\ttfamily LOBBY}\+: Connected, waiting for a game
\item {\ttfamily INGAME}\+: Participating in an active game
\item {\ttfamily SPECTATING}\+: Observing a game
\end{DoxyItemize}
